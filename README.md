# Ops OS

A case file showing an AI operating layer built for a carbon farming ops team.

## View the showcase

Live page: **https://cbryant9.github.io/ops-os-showcase/**

## What's in here

- `index.html` - the showcase page (Swiss-editorial styling, single-file build)
- `screenshots/` - supporting imagery used in the page

## About

Ops OS documents the agent stack, workflows, and operating layer running behind a real-world carbon farming operations team. It's presented as a case file rather than a sales page - the goal is to show how the system is wired together, not to sell it.
